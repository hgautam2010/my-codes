## Approach

The solution reverses the input string and iterates through it, appending alphanumeric characters to the result string while inserting dashes after every k characters. Characters are converted to uppercase before appending. Finally, the result string is reversed to produce the correctly formatted license key.

## Complexity

- Time: O(n)
- Space: O(n)
