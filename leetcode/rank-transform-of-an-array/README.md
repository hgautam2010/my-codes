## Approach

The solution assigns ranks to elements in the array based on their sorted order with duplicates receiving the same rank, using sorting and mapping indices.

## Complexity

- Time: O(n log n) due to sorting
- Space: O(n) for auxiliary storage of element-index pairs

- Sorting
- Ranking
- Index Mapping
