## Approach

This solution uses a two-pointer technique to duplicate zeros in-place within the array. It first calculates the position where elements would be if the array were extended to include duplicates, then shifts and copies elements backward to avoid overwriting values prematurely.

## Complexity

- Time: O(n)
- Space: O(1)
